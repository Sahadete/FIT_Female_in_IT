Desktop Support Technicians

Desktop Support Technicians job description includes: assisting staff with technical support of desktop computers, applications, and related technology. The support includes specification, setting up, and testing of computer systems and peripherals within established standards and guidelines. The position profile is applicable for both ICT Companies and non-ICT Companies.

  1. Desktop Support Technician Courses:
    • ECDL-European Computer Drivers License
    • Windows Operating System Fundamentals
    • Windows Server Administration Fundamentals
    • Security Fundamentals
    • IT Essentials: PC Hardware & Software
    • CCNA 1: Introduction to Networks
    • CCNA 2: Routing and Switching Essentials
    • Elementary and Intermediate English
    • CV Writing,Interview,Presentation Skills etc.

Network and System Administrators

On a global context, employment of network and computer systems administrators is projected to grow 12 percent from 2012 to 2022, about as fast as the average for all occupations. Growth will be highest at firms that provide cloud computing technology. Network and computer systems administrators work with the physical computer networks of a variety of organizations and therefore are employed in many industries.

  1. System & Network Administrator Courses
    • CCNA 1: Introduction to Networks
    • CCNA 2: Routing and Switching Essentials
    • CCNA 3: Scaling Networks
    • CCNA 4: Connecting Network
    • Configuring Windows 8
    • Installing and Configuring Windows Server 2012
    • Administering Windows Server 2012
    • Configuring Advanced Windows Server 2012
    • Elementary and Intermediate English
    • CV Writing,Interview,Presentation Skills etc.

Software and Application Developers

The program is multidimensional and candidates might not sit all trainings, but chose those applicable to their passion. Applications developers translate software requirements into workable programming code and maintain and develop programs for use in business. Most will specialize in a specific development field - such as mobile phone applications, accounting software, and office suites or graphics software - and will have in-depth knowledge of at least one computer language.
